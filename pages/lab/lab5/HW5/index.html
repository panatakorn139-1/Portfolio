Underline text animation

--start: 0.1em; --color: #000; color: #ffffff00;
padding-bottom: var(--start);
background: linear-gradient(90deg,var(--color) 50%,#000 0)
calc(100% - var(--_p,0%))/200% 100%,
linear-gradient(var(--color) 0 0)
0% 100%/var(--_p,0%) var(--start) no-repeat;
-webkit-background-clip: text,padding-box;
background-clip: text,padding-box;
transition: 0.5s;
And add pseudo code with hover.
:hover{--_p: 100%}
