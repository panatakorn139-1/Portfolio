Lasson

ผมได้เรียนรู้การวางแผนและออกแบบ, การออกแบบ Graphic และimage, ได้เรียนรู้ HTML และCSS, การใช้ Responsive Design การให้เว็บไซต์เป็น Responsive เว็บไซต์จะปรับขนาดและรูปแบบตามอุปกรณ์ที่ผู้ใช้ใช้งาน เพื่อให้ผู้ใช้สามารถเข้าถึงได้ทุกแพลตฟอร์ม การออกแบบ UX/UI รวมถึงการใช้ Framework เช่น jQuery Bootstrap Tailwind การทำงานเป็นทีม การสร้างผลงานและทักษะอาชีพ นำมาใช้ในการสร้างอาชีพ เรียนรู้ประวัติเกี่ยวกับของการออกแบบเว็บไซต์ ถึงอาจารย์จะไม่ได้สอน javascript แต่ผมก็พยายามศึกษามัน จนกลายเป็นว่าผมชอบกับการออกแบบและเขียนเว็บมากเลยล่ะครับ
